✦ Help & Updates

iClass only supports one scoring system per account. When gymnastics switched to the new 1-star key-component system, other programs like ninja and tumbling can no longer be evaluated on the 5-star system — they're the same as gymnastics now, a yes/no on each criteria.

You're more than welcome to enter your own criteria under the new star system that fits each skill — whatever works for your program.

If you'd like to bounce ideas around, reach out to Jayme or Kim anytime.

The converter currently reads and converts: Preschool, Junior, Advanced Junior, Level 1, Level 2, Level 3.

Ninja, Tumbling, Cheer, IGT, and Specialty don't convert — print those the old way in iClass.

💡 If a program gets skipped, the converter will tell you — so you'll always know what made it in and what didn't.

Use this format: Program | Day | Time | Ages (note)

The converter reads each section between the pipes to build the PDF. If a note like "Class Closing 5/14" comes before the program name, everything shifts out of place.

Put the note at the end in parentheses instead:

Level 1 | Thursday | 3:30pm | Ages 7+ (Class Closing 5/14)

Pick one, or combine them. Each option adds one page per class.

  • ALL 4 EVENTS — classic single page, everything together.
  • VAULT + BARS — focused sheet, bigger text.
  • BEAM + FLOOR — focused sheet, bigger text.

Click more than one to get a multi-page set. (e.g. ALL 4 + VAULT+BARS = 2 pages per class: overview + focus.)

When your PDF finishes, you'll see a new Convert Another button on the download screen. Drop the next file without leaving your gym's modal — no need to re-pick your gym every time.
