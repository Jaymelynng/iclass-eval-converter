3

4
iClassPro Select XLS format and download the report to your computer

5
evalconvert.com Go to evalconvert.com and select your gym
6
evalconvert.com Drop the XLS file — review the class list that populates
7

🛠 If something's off
📄
File won't upload?
Must be XLS — not PDF or CSV. Go back to Step 4 and choose XLS when exporting.
❓
Missing a class?
Class name needs a program keyword (Preschool, Junior, Level 1, etc). Classes without one are silently skipped.
📝
Weird class name on PDF?
Rename in iClassPro to match: Program | Day | Time | Ages
📑
Extra pages?
Over-capacity classes (more than 6 preschool / 8 level) auto-split into (pg 1/2). Normal — print both.
