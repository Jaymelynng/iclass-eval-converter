How to Export & Print Evaluation Sheets

Part 1 — iClassPro
1
iClassPro Log into iClassPro and click Classes in the top navigation bar
2
iClassPro Filter by program, then select the specific day you want to print
3
iClassPro Select all classes, then click Reports → Class Evaluation Forms
4
iClassPro Select XLS format and download the report to your computer
Part 2 — evalconvert.com
5
evalconvert.com Go to evalconvert.com and select your gym
6
evalconvert.com Drop the XLS file — review the class list that populates
7
evalconvert.com Click Download PDF — your branded eval sheets are ready

How the scoring works
One scoring system. All of iClass.
iClass only runs one scoring system at a time for your account. When gymnastics moved to the new 1‑star, key‑component system, every program in iClass flipped with it. This isn't something we can split per program.
If something's off
1
File won't upload
Must be XLS — not PDF or CSV. Go back to Step 4 and choose XLS when exporting.
2
Missing a class?
Class name needs a program keyword (Preschool, Junior, Level 1, etc). Classes without one are silently skipped.
3
Weird class name on PDF?
Rename in iClassPro to match: Program | Day | Time | Ages
4
Extra pages?
Over-capacity classes auto-split into (pg 1/2). Normal — print both.
